# 🤖 Discord Captcha Botu

[![Discord.js](https://img.shields.io/badge/Discord.js-v14-blue.svg)](https://discord.js.org)
[![Node.js](https://img.shields.io/badge/Node.js-16.x-green.svg)](https://nodejs.org)

[Pulkadot](https://discord.gg/aTNjPQd2E6) tarafından geliştirilmiş, captcha güvenlik sistemine sahip Discord botu.

## 📌 Özellikler

- 🎯 **Captcha Doğrulama Sistemi**
  - Sunucuya yeni katılan kullanıcılara otomatik captcha gönderimi
  - Belirlenen sürede doğrulama yapılmayan kullanıcıların banlanması
  - Özelleştirilebilir captcha tasarımı

- ⚙️ **Komut Sistemi**
  - `/ping` - Botun gecikme süresini kontrol etme
  - `/sistemi aç` - Captcha'yı aktifleştirme
  - `/sistemi kapat` - Captcha'yı devre dışı bırakma

### 🚀 Kurulum gereksinimleri
